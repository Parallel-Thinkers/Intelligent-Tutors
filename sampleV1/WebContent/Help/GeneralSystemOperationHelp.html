General System Operation Help

  1. When you login in the system, you need to select a type of problem which you want to solve or practice or learn. After selecting the type of problem you need to select the level (easy, medium, and hard) so that system will display a problem of the selected type and selected level.
  2. The Illustration tab shows how to use the system and type the text in solving mode with an example.
  3. Once the problem is displayed as per your selection then you can either select Learning Mode or Solving Mode. If you feel that you want to solve the problem directly without learning it then you can skip the learning mode and go directly to the solving mode.
  4. By clicking on the tab My Learning Profile you can see what you have learned so far.
  5. By clicking on Home tab you can return to home page where problem types and level are listed.
  6. By clicking Log Out tab you can completely log out of the system.
