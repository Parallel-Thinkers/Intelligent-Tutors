Text entering help

  1. Names of the real world objects should be written as the way they are given in the problem. For confirmation you can see the Semantic Knowledge tab where the name of the object would have been displayed.  
  2. In this syntax in the LHS statement the dot in between the object and its measurement is compulsory and the unit should be written in the short form like Cm without giving any space. In the RHS statement the space between value and unit (again short form) is mandatory. For example: If the real world object is a cuboid water tank then its length given in centimeter can be written in the following way, WaterTank.LengthCm = 2.5 Cm  
  3. The letters can be all small letters except the beginning of a unit. Unit must be written like Cm, M, Gm, Kg, Liter, Nos. etc.
  4. If it is asked to find volume of the water tank then you would enter the following text in the To Find: WaterTank.VolumeCuCm = ?

    If you observe RHS of the above statement contains no value and unit,instead it has a question mark.  

    The calculated answers should be correct up to two decimal places. For example if the calculated answer is: 34.562 then you should write the same. The iterative answers must be rounded up to two digits. For example if the calculated answer is: 32.3333 then you must round up the digits and write correct up to two decimal places i.e. 32.34.
