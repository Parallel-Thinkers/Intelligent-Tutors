Learning Mode Help

  1. Learning Mode selection will take you to tutoring module where you will be asked analytical questions that will help you to analyze a problem. This will provide training for how to think to solve any given problem.
  2. Questions will be displayed at the right side of the problem and there will be a drop-down box which will provide the choices of possible answers. You can select the correct answer. For every wrong answer there is a constructive feedback which will help you to think about the correct answer. After selection of correct answer you will get a feedback message and after clicking that message the next question will be displayed.
  3. The list of questions and correct answers so far you have learned in that particular session will be displayed in the part below the question and choices drop-down box which you can refer to as a part of thinking process.
  4. If you are in Learning Mode of a particular problem and want to re-learn a problem then you can click on the tab Start Over in the learning mode which will bring the system into the original position of the Learning Mode of that particular problem and you can start learning the problem again.
  5. While in learning mode if you feel that you have learned the required content and you do not need further learning then you can go to solving mode and start solving the problem. If in between the solving if you feel like coming back to learning mode you can click on the Learning Mode tab and return to the point where you left the learning of that particular question.
  6. If you complete learning of a problem then all the questions asked by the system and the correct answers to those questions will be displayed in a pop-up window. This you can refer to as a part of thought process development while solving the problem.
  7. If you select solving mode directly and complete the problem then you will not be able to go back to Learning Mode of that particular problem as you already know how to solve that problem. However, if you are in between the solving mode and if you want to return to Learning Mode to learn the problem then you can click on the Learning Mode tab and go to Learning Mode.
